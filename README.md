# PyVehicle
**Python Vehicle Code**

**Python3 Packages**
Python packages required: json, pycrate, pyside2, scapy, ...

**Linux Installation** Installation:
cd packages
./packages.sh

Add classes and classes/j2735 to your PYTHONPATH: export PYTHONPATH=:./classes:./classes/j2735

**Windows Installation**
cd packages
.\packages.cmd
